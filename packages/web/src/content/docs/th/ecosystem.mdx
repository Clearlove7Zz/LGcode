ชุดโครงการชุมชนที่สร้างขึ้นบน LGcode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ LGcode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-lgcode](https://github.com/awesome-lgcode/awesome-lgcode) และ [lgcode.cafe](https://lgcode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
