ชุดโครงการชุมชนที่สร้างขึ้นบน Loongcode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Loongcode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-loongcode](https://github.com/awesome-loongcode/awesome-loongcode) และ [loongcode.cafe](https://loongcode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
